# 🤖 부비라이브 멀티 모니터링 자동화 봇 (Bubi Auto Server)

부비라이브(Bubeelive) 방송국의 채팅 및 후원 이벤트를 24시간 실시간으로 감시하고, 설정된 명령어 매크로에 자동으로 답변하는 모니터링 서버 패키지입니다.

---

## 💻 1. 내 컴퓨터(로컬 PC)에서 실행하는 방법

깃허브 저장소를 그대로 다운로드하여 어느 PC에서든 쉽게 실행할 수 있습니다.

### 사전 준비
* [Node.js 공식 홈페이지](https://nodejs.org/)에서 **LTS 버전**을 다운로드하여 설치합니다.

### 실행 방법
1. 저장소 파일들이 있는 폴더로 들어갑니다.
2. **`run.bat`** 파일을 더블 클릭하여 실행합니다.
3. 처음 실행하면 `.env` 파일이 생성되면서 메모장에 적으라는 경고가 뜹니다.
4. 폴더 내 새로 생긴 **`.env`** 파일을 메모장으로 열어 다음 값을 입력하고 저장합니다:
   * `BUBEE_ID`: 로그인할 부비라이브 계정 아이디
   * `BUBEE_PW`: 계정 비밀번호
   * `BUBEE_ROOM_ID`: 모니터링할 방 번호 (여러 방인 경우 쉼표로 연결: 예: `115654,115656`)
   * `BUBEE_COOKIES`: (선택) 해외 서버 또는 신규 IP 로그인 캡차 우회용 쿠키 JSON (아래 설명 참고)
5. 다시 **`run.bat`**을 더블 클릭하면 자동으로 패키지를 설치하고 봇이 켜집니다.

---

## ☁️ 2. 클라우드 서버(Railway 등)에서 실행하는 방법

1. 이 프로젝트 코드를 본인의 **개인 깃허브 저장소(GitHub Repository)**에 업로드합니다.
2. [Railway](https://railway.app/)에 로그인하여 깃허브 저장소를 연동시킵니다.
3. Railway 프로젝트의 **`Variables` (환경 변수)** 탭에서 다음 변수들을 추가합니다:
   * `BUBEE_ID`: 부비라이브 아이디
   * `BUBEE_PW`: 부비라이브 비밀번호
   * `BUBEE_ROOM_ID`: 방 번호 (예: `115654,115656`)
   * `BUBEE_COOKIES`: 브라우저 복사 쿠키 JSON (필수 권장)
4. 저장하면 자동으로 빌드 및 배포가 완료되며 24시간 무중단 가동됩니다.

## 🍪 3. 캡차(CAPTCHA) 로그인 우회용 쿠키 추출법

서버나 다른 PC에서 로그인할 때 보안 문자(캡차)로 막히는 문제를 우회하는 방법입니다.

1. 크롬 브라우저로 부비라이브에 로그인합니다.
2. 키보드 **`F12`**를 눌러 개발자 도구를 켜고 **`Console` (콘솔)** 탭으로 이동합니다.
3. 아래 코드를 복사해서 붙여넣고 **`Enter`**를 칩니다.
   ```javascript
   console.log(JSON.stringify(document.cookie.split('; ').map(c => { const [name, ...val] = c.split('='); return { name: name.trim(), value: decodeURIComponent(val.join('=')), domain: '.bubeelive.com', path: '/' }; })));
   ```
4. 바로 아래 줄에 출력된 대괄호 `[{"name": ...}]` 로 시작하는 긴 텍스트 전체를 복사합니다.
5. 설정 파일(`.env` 또는 Railway 환경 변수)의 **`BUBEE_COOKIES`** 항목에 붙여넣고 저장합니다.

## 🛠️ 주요 커스터마이징 (명령어 수정)
* **기본 설정 및 매크로 수정**: `userscript.js` 파일을 메모장이나 에디터로 열어 수정 후 다시 깃허브에 올리거나 로컬에서 재시작하시면 변경 사항이 즉시 적용됩니다.
